# Фронтенд дипломной работы для Яндекс.Практикум
Дипломная работа - сервис, в котором можно найти фильмы по запросу и сохранить в личном кабинете

## Используемые технологии
1. HTML5
2. Методология БЭМ
3. Флексбокс-верстка
4. Grid-css
5. Отзывчивая верстка
6. React
7. Git
